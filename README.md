# Plant vs Zombie

一个使用原生 ESM JavaScript 和 Three.js 构建的植物大战僵尸风格网页项目，当前聚焦于 PC 浏览器下的单机关卡与无尽模式体验。

## 当前内容

- 5x9 草坪战场与 3D 透视渲染
- 选卡、种植、铲子、暂停、倍速、镜头切换
- 波次关卡与无尽模式
- 阳光经济、卡牌独立冷却、局内继续存档
- 对象池、小推车、结算复盘
- 多种植物与僵尸单位，包含部分特殊机制

## 运行方式

这是一个纯前端项目，不依赖构建工具。使用任意静态服务器打开项目根目录即可。

示例：

```powershell
cd D:\dev\plant-vs-zombie
python -m http.server 8080
```

然后在浏览器访问：

```text
http://localhost:8080
```

## 目录结构

- `index.html`：页面入口
- `assets/vendor/`：本地第三方依赖
- `docs/`：需求文档
- `src/config/`：植物、僵尸、关卡等配置
- `src/core/`：状态、存档、音频
- `src/game/`：战斗、事件、流程、实体更新
- `src/render/`：Three.js 渲染
- `src/ui/`：界面与面板逻辑

## 已实现的代表单位

- 植物：向日葵、豌豆射手、坚果墙、高坚果、土豆雷、窝瓜、地刺、樱桃炸弹、寒冰射手、双发射手、食人花
- 僵尸：普通、旗帜、路障、铁桶、撑杆、读报、铁门、橄榄球、舞王、伴舞
